git add .

## How to run?

```bash
conda create -n safety python=3.8 -y
```

```bash
conda activate safety
```

```bash
pip install -r requirements.txt # Copied from YOLOv7 official documentation. -e . is used for setup.py installation
```

```bash
python app.py
```
